# Sealed Letter

Private notes, protected by your own passkey.

- No account, no login
- End‑to‑end encrypted in your browser
- Works on any device

## How it works
1. Write a note in your browser.
2. Set a passkey (a phrase you’ll remember).
3. The note is encrypted and stored in Firebase.
4. Later, on any device, type the same passkey to decrypt and read it.

## Technology
- Front‑end: HTML, CSS, JavaScript
- Encryption: Web Crypto API (AES‑GCM, PBKDF2)
- Storage: Firebase Realtime Database
